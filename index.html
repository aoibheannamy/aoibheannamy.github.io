This web application attempts to answer how film and television conveys hidden meanings and emotions effectively by breaking down scenes into important parameters.

Data from Amazon's Mechanical Turk labelling, along with manual labelling, is stored here for easy access.

The effectieness of the gathered metadata is tested in a visual novel representation.

This web application was created in partial fulfillment of Masters of Software Development at Queen's University.
